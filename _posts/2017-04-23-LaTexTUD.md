### Forword
Whenever I reinstalled my Mac OS, or my friends asked me to help them with the installation of LaTex with TUD-Design, it has always been a pain in the ass.     

### Step_01: Get the MacTeX (TeX Live).
This is a very simple step. You have two options to install it. 

> * The first option is to download MacTeX directly from [The TeX Users Group](http://www.tug.org/mactex/). 

Or

> * You can download MacPorts from [here](http://www.macports.org/), and then install texlive using command line. 
* ~~~~
$ sudo port selfupdate
$ sudo port -d install texlive
~~~~

### Step_02: Get the *tuddesign* and *tudfonts*. 
After successfully installing the TeX Live, we need to download two zip files from [TUD-Design website](http://exp1.fkp.physik.tu-darmstadt.de/tuddesign/), *tudfonts.zip* and *tuddesign.zip*. Open the terminal, and type in the following commands: 

~~~~
$ cd /path/to/zip/files/ (e.g.: cd ~/Downloads/)
$ unzip latex-tuddesign_current.zip -d ~/Library/
$ unzip tudfonts-tex_current.zip -d ~/Library/
~~~~ 
The first command is to go to the directory where you save those two zip files. The second and third command will unzip the files, creat a new folder called *texmf* in *~/Library/* directory, and put its content in *~/Library/texmf/*. Then put these command in your terminal, I don't know very specifically how does it work, just do it.  

~~~~
$ sudo mktexlsr
$ sudo updmap-sys --enable Map 5ch.map
$ sudo updmap-sys --enable Map 5fp.map
$ sudo updmap-sys --enable Map 5sf.map
~~~~ 

### Step_03: Setup TeXstudio. 
You will have many options of choosing a writing environment for creating LaTeX documents. I personally perfer TeXstudio, because it has numerous features like syntax-highlighting, integrated viewer, reference checking and various assistants. For downloading and more details see the [TeXstudio](http://www.texstudio.org/). 

After installing it, please open it and go to *TeXstudio -> Preferences -> Commands*, set the commands directory like this.  For *build*, you can set the *Default Compiler* as *PdfLaTex*.

<img src="/images/posts/MacOS/TeXstudio.png" height="200" width="800">  

And now you are good to go and use the LaTeX template from TU Darmstadt.
