**Artificial Intelligence** has been achieving great successes in recent years, e.g., natural language processing or understanding, game playing, computer vision, ect. Among all the state-of-the-art approaches, __reinforcement learning__ is a type of unsupervised machine learning, which allows agents to automatically determine the optimal policy based on feedback from the enviornment, in order to maximize the cumulative reward.

### Reinforcement Learning 
This is the description of a basic framing of a reinforcement learning scenario from wikipedia, in which a agent explores the enviornment throung a set of actions or policies and retrives the representations of a interpreted reward and state. The agent has to decide the optimal policy (actions) to collect as much reward as possible.  
<img src="/images/posts/RL_intro/agent.png" height="300" width="290">  
__Sequential Decision Making__ : The ultimate goal is to select actions to maximise total future reward. A reinforcement learning agent interacts with its environment in discrete time steps. At each time $t$, the agent receives an observation $ o_t$, which typically includes the reward $r_t$. It then chooses an action $a_t$ from the set of actions available, which is subsequently sent to the environment. The enviornment moves to a new state $s_{t+1}$ and the reward $r_{t+1}$ associated with the transition $ (s_t, a_t, s_{t+1} ) $ is determined. However, actions may have long term consequences and the reward may be delayed. It may be better to sacrifice immediate reward to gain more long-term reward,  e.g., a financial investment may take months to mature.  

__Enviornment__ : The enviornment can be fully observable or partially observable. In fully observable enviornment senario, we assume the agent knows all the states and the interprated reward for a given action. Whereas in the opposing case, the agent can only observe its surrounding enviornment.

__Major Components of an Reinforcement Learning Agent__: 

* Policy: agent's behaviour function, e.g., stochastic policy: $$\pi(a \arrowvert s) = \mathbb{P}[A_t = a \arrowvert S_t = s]$$
* Value Function: how good is each state and / or action, e.g.,$$v_{\pi}(s) = \mathbb{E}_{\pi} [R\_{t+1} + \gamma R\_{t+2} + \gamma^2 R\_{t+3} + ... \arrowvert S\_t = s]$$
* Model: agent's representation of the enviornment, it predicts what the enviornment will do next, e.g., 
$$P_{ss'}^a = \mathbb{P} [S_{t+1}=s'|S_t = s, A_t=a]$$

### Big Picture

The RL agent can be categorized in two different ways. One way is to classify the agent based on the value funtion and policy.

> * Value Based
	* No policy
	* Value Function
* Policy Based
	* Policy
	* No Value Function
* Actor Critic
	* Policy
	* Value Function

Another way is to classify it based on the model. 

> * Model Free 
	* Policy and /or Value Function
	* No Model
* Model Based
	* Policy and /or Value Function
	* Model

Exploration vs. Exploitation (it is important to find a trad-off between these two strategies):
> * Exploration: finds more information about the enviornment
* Exploitation: exploits known information to maximise reward. 

### Applications

**AlphaGo:**
AlphaGo is an artificial intelligence program developed by Google DeepMind to play the board game Go. In March 2016, it beat Lee Sedol in a five-game match, the first time a computer Go program has beaten a 9-dan professional without handicaps. According to the paper published in Nature, AlphaGo relies on two different components, a tree search produce, and convolutional neural networks that guid the tree search procedure. To train the AlphaGo, they also use *Deep reinforcement learning*, the policy networks were therefore improved by letting them play against each other. 
<img src="/images/posts/RL_intro/alphago.jpg" height="350" width="300">  

**Fly stunt manoeuvres in a helicopter:** Stanford computer scientists have developed an artificial intelligence system that enables robotic helicopters to teach themselves to fly difficult stunts by watching other helicopters perform the same maneuvers. The algorithm is developed by Pieter Abbeel, Andrew Y. Ng (吴恩达)，which called *Apprenticeship learning via inverse reinforcement learning*.
