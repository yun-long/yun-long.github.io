### Single Chain problem
In this example, we will try to solve the single chain problem with deep reinforcement learning method. First, let's take a look at what is single chain problem. The single chain environment contains **2 actions** and **5 states**. When the agent take an action of 0, the agent will be sent to next state and receive a reward of 0 before it reaches state 4 and a large reward of 10 when it reaches state 4. The agent will return to state 0 whenever it takes an action of 1 and receive a small reward of 2.

### Policy Network
First, we define a three layers neural network to represent the policy. The input of the network has 5 neurons which takes 5 states as inputs. It has 2 outputs which predict the probabilities of taking actions given a state. To describe the policy network with math equations, 

$$
\begin{eqnarray}
\mathbb{P}(\boldsymbol{a} \mid \boldsymbol{s} ) &=& \pi(\boldsymbol{a} \mid \boldsymbol{s, W1, W2} ) \\
&=& \text{softmax} (\text{ReLu}\boldsymbol{(s * W1)} * \boldsymbol{W2}) 
\end{eqnarray}
$$

The goal of the agent is to find a policy $\pi$ that maximizes the expected return,

$$
\begin{eqnarray}
\text{max} J(\pi) = \sum_{s, a} \pi(a \mid s) r(s, a)
\end{eqnarray}
$$

A python/numpy implementation of this policy can be written as below, where **$W1$** and **$W2$** are two matrices that we initialize randomly using Xavier initialization. We use softmax function at the end, which squashes the output probability to the range [0, 1]. Intuitively, the neurons in the hidden layer are the features of the state, and the weights in $W2$ can then decide which action to take according to the state features. Now, the question is how can we find the optimal parameters $\hat{W1}$, $\hat{W2}$ for the policy network, that enables the agent to accumulate rewards as much as possible.

def softmax(x):
    e_x = np.exp(x - np.max(x))
    return e_x / e_x.sum(axis=0)

    x[x<0] = 0
